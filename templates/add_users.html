Add New Users:

Enter the new users in the following format:

user1, pass1
user2, pass2

Make sure to put the username, followed with a comma, followed with the password. Don't worry about whitespace on the end of the line/between words, it will be trimmed.
Make sure to make a new line for each user. The form will tell you if there were formatting issues and if the input was succesful. You should double check to see if they were added.
I would recommend making a text document with the input correctly formatted, then copy/pasting it here. This will make life easier if something goes wrong, since the text will dissapear after submission.
